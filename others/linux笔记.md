linux笔记

## 基本指令

```
uname -a            # 显示环境信息
ps -ef|grep python  # 搜索进程
kill -9 ID          # 结束进程
sed -i              # 删除文本操作
du -h --max-depth=1 # 查看内存占用情况

```


### grep常用命令

```
-A  # 除了显示符合范本样式的那一行之外，并显示该行之后的内容
-b  # 在显示符合范本样式的那一行之外，并显示该行之前的内容
-B  # 除了显示符合样式的那一行之外，并显示该行之前的内容
-c  # 计算符合范本样式的行数
-C  # 除了显示符合范本样式的那一行之外，并显示该行前后的内容
-d  # 当指定要查找的是目录而非文件时，必须使用这项参数，否则grep命令将回报信息并停止动作
-e  # 指定字符串作为查找文件内容的范本样式
-F  # 将范本样式视为固定字符串的列表
-G  # 将范本样式视为普通的表示法来使用
-h  # 在显示符合范本样式的那一列之前，不标示该列所属的文件名称
-H  # 在显示符合范本样式的那一列之前，标示该列的文件名称
-i  # 忽略字符大小写的差别
-l  # 列出文件内容符合指定的范本样式的文件名称
-L  # 列出文件内容不符合指定的范本样式的文件名称
-n  # 在显示符合范本样式的那一列之前，标示出该列的编号
-q  # 不显示任何信息
-s  # 不显示错误信息
-v  # 反转查找
-w  # 只显示全字符合的列
-x  # 只显示全列符合的列
-o  # 只输出文件中匹配到的部分
^   # 匹配以XX开头的行
$   # 匹配以XX结尾的行
```

## vim 文本编辑器使用

```
i		写
esc		退出写
:q		关闭窗口
:w		保存
:set nu	打开行号
:set nu!	关闭行号

多行添加同样的内容
1.vim 打开文件，并移动光标到要添加列的起始行
2.按下ctrl+v，打开visual模式
3.通过光标向下选中你要添加内容的位置
4.按下I（即shift＋i）键，然后输入你要插入的内容
5.按下ESC键，大概1s后，你就能看到内容加上了
```

## 指令组合

```
不区分大小写统计 aaa.txt 文件中 a 出现的次数
grep 查询出匹配到的字符，传递给 wc 统计出次数 -l是统计行数
grep -o -i 'a' aaa.txt | wc -l

删除空文件夹
find -type d -empty | xargs rm -rf

递归删除
find . -name "*.pb.*"  | xargs rm -f

显示文件数量
ls -lR| grep "^-" | wc -l

查找并删除文件
rm -f `ls | grep -irn "XXXXXXXX" -l`

批量删除文件
find -name "*.dot" -exec rm -f '{}' \
find ./ -name "*.pyc" | xargs rm

```
